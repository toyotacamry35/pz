There is an extended version of the demo scene available.  
To install this please follow this steps. 

1. Install the Environments asset package included with Unity. In the menu select Assets/Import Package/Environments. You can install all of the package but at a minimum the folder with the free speedtrees. 

2. go to the website https://www.awesometech.no/index.php/vegetation-studio-download-extended-demo/
   Download the demo package on this page and install in the project.		

3. Open the scene DownloadedDemo.scene in the folder AwesomeTechnologies/VegetationStudioDownloadedDemo
