Introduction

The Ultimate Mesh Combiner is a Unity extension that allows you to select multiple
objects and bake them into one single GameObject.
The Mesh Combiner supports static meshes with different properties (normals,
uv channels, vertex colors...).


________________________________________________________________________________________
Requirements

Supports Unity 3.5+ and 4.x
